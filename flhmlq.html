FLHLMQ

Web Development | Programming

Process and intention

In our Projet Web class, we were tasked with redesigning the website for Fédération des locataires d'HLM du Québec. Our team focused on creating a bilingual and responsive site using WordPress to ensure it worked well across various devices. We implemented a multilingual plugin to make the site accessible to both French and English-speaking users and selected a responsive theme to guarantee it displayed properly on mobile devices. The objective was to provide an intuitive user experience with easy navigation, clear calls-to-action, and accessible resources for tenants. Through this project, I gained hands-on experience in customizing WordPress themes, implementing responsive design, and improving overall user experience. This project has not only deepened my web development skills but also sparked my interest in further exploring UX/UI design, multilingual content, responsive design, WordPress customization, and front-end development in future projects.
